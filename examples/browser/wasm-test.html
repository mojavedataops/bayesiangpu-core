Testing WebGPU Backend

The default build uses CPU (NdArray) backend. To test WebGPU:

  1. Build GPU version: cd js && npm run build:wasm:gpu
  2. Start local server: cd examples/browser && python -m http.server 8080
  3. Open Chrome 113+ (WebGPU enabled by default)
  4. Navigate to http://localhost:8080/wasm-test.html
