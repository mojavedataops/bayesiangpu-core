Poisson Count Model

Model

Estimating an unknown event rate from observed count data.

Model:
  counts_i ~ Poisson(rate)        [likelihood]
  rate ~ Gamma(2, 0.5)            [weakly informative prior]

The conjugate posterior is Gamma(2 + sum(y), 0.5 + n), but we use
MCMC via BayesianGPU's generic run_inference() to demonstrate
the general-purpose sampler.

Synthetic data: 30 observations from Poisson(rate=4.5)
