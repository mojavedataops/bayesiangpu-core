# Log-Normal Distribution

Create a Log-Normal distribution. For positive values with
multiplicative effects.

## Usage

``` r
LogNormal(loc, scale)
```

## Arguments

- loc:

  Mean of the log (mu)

- scale:

  Standard deviation of the log (sigma)

## Value

A distribution specification

## Examples

``` r
if (FALSE) { # \dontrun{
LogNormal(0, 1)
} # }
```
