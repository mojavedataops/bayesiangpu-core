# Logistic Distribution

Create a Logistic distribution. Similar shape to Normal but with heavier
tails, related to the logistic function.

## Usage

``` r
Logistic(loc, scale)
```

## Arguments

- loc:

  Location parameter (mean)

- scale:

  Scale parameter (must be positive)

## Value

A distribution specification

## Examples

``` r
if (FALSE) { # \dontrun{
Logistic(0, 1)  # Standard Logistic
} # }
```
