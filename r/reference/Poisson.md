# Poisson Distribution

Create a Poisson distribution for count data.

## Usage

``` r
Poisson(rate)
```

## Arguments

- rate:

  Rate parameter (expected count, can be a number or parameter name)

## Value

A distribution specification

## Examples

``` r
if (FALSE) { # \dontrun{
Poisson(5)          # Rate of 5
Poisson("lambda")   # Parameter from model
} # }
```
