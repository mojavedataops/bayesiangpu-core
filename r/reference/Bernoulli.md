# Bernoulli Distribution

Create a Bernoulli distribution for binary outcomes (0 or 1).

## Usage

``` r
Bernoulli(p)
```

## Arguments

- p:

  Probability of success (can be a number or parameter name)

## Value

A distribution specification

## Examples

``` r
if (FALSE) { # \dontrun{
Bernoulli(0.5)       # 50% success rate
Bernoulli("theta")   # Parameter from model
} # }
```
