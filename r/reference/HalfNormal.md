# Half-Normal Distribution

Create a Half-Normal distribution (positive values only). Useful for
scale parameters.

## Usage

``` r
HalfNormal(scale)
```

## Arguments

- scale:

  Scale parameter

## Value

A distribution specification

## Examples

``` r
if (FALSE) { # \dontrun{
HalfNormal(1)  # Half-normal with scale 1
} # }
```
