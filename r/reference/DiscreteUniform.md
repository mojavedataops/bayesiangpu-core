# Discrete Uniform Distribution

Create a Discrete Uniform distribution. Each integer value in \[low,
high\] has equal probability.

## Usage

``` r
DiscreteUniform(low, high)
```

## Arguments

- low:

  Lower bound (integer)

- high:

  Upper bound (integer)

## Value

A bg_distribution object

## Examples

``` r
if (FALSE) { # \dontrun{
DiscreteUniform(1, 6)  # Fair die
} # }
```
