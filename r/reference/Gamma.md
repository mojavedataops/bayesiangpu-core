# Gamma Distribution

Create a Gamma distribution. Defined on (0, infinity), useful for
positive parameters.

## Usage

``` r
Gamma(shape, rate)
```

## Arguments

- shape:

  Shape parameter (k)

- rate:

  Rate parameter (1/scale)

## Value

A distribution specification

## Examples

``` r
if (FALSE) { # \dontrun{
Gamma(2, 1)  # Shape=2, rate=1
} # }
```
