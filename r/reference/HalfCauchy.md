# Half-Cauchy Distribution

Create a Half-Cauchy distribution (positive values only). Heavy-tailed
prior for scale parameters, widely recommended for hierarchical models.

## Usage

``` r
HalfCauchy(scale = 1)
```

## Arguments

- scale:

  Scale parameter (default: 1)

## Value

A distribution specification

## Examples

``` r
if (FALSE) { # \dontrun{
HalfCauchy(5)  # Half-Cauchy with scale 5
} # }
```
