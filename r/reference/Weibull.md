# Weibull Distribution

Create a Weibull distribution. Commonly used for survival analysis and
reliability modeling.

## Usage

``` r
Weibull(shape, scale)
```

## Arguments

- shape:

  Shape parameter (k \> 0)

- scale:

  Scale parameter (lambda \> 0)

## Value

A bg_distribution object

## Examples

``` r
if (FALSE) { # \dontrun{
Weibull(1.5, 1)  # Shape=1.5, scale=1
} # }
```
